## Description


This commandline application will help any user to determine the amount that they will get after the given period at the
interest rate depending on the selection of the type of interate selected ( Simple or Compound Interest ). This application will also help the user calculate the home loan repayment

## How To Use

#### Requirement of Use

- Python must be installed before use, and any other code editor that supports python can be used to run the application



#### Instructions

- Make a selection between Investment and Home Loan
- If investement. Enter the amount of money that you are depositing as well as the following.
  - The interest rate (as a percentage)
  - The number of years you plan on investing for
  - input whether you want “simple” or “compound” interest
 
- If the user selects ‘bond’, then enter the following
  - The present value of the house. E.g. 100000
  - The interest rate. E.g. 7
  - The number of months you plan to take to repay the bond. E.g. 120
